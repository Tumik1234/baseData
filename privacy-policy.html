Our Bot may collect and process the following data: Discord identifiers: such as server ID (guildId), user ID, and channel ID. Server configuration data: including selected log channel, bot language, and log message status. Message content (text and images), only for safety checks (e.g., detecting inappropriate content). Image URLs if included in user messages. Image analysis results from Google Cloud Vision API (e.g., flags for adult, racy, or violent content). 🔹 How is the data used? The data is used exclusively to: provide the Bot’s core features (event logging, content moderation), enhance community safety by removing potentially harmful images, allow server admins to configure the Bot. 🔹 Is data stored? Yes. Server configuration data is saved locally in a file named guildSettings.json. Content such as message texts and images is not permanently stored — it is only used in real-time for moderation purposes. 🔹 Is data shared with third parties?
